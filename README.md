# Fizyr Objection Roulette

A browser-based sales-training app for practising responses to common
Fizyr / Fizyr OS objections. Spin a question, answer it (voice or typed),
then score and reveal coaching.

Live: https://tmoyo.github.io/fizyr-objection-roulette/

> Voice capture uses the browser Web Speech API and works in Chrome or Edge
> over https. The typed-answer fallback works everywhere.
